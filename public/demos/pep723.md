# PEP 723 dependencies

This document declares its dependencies with [PEP 723 inline script
metadata](https://peps.python.org/pep-0723/) in its **first code cell** (the
same convention [`juv`](https://github.com/manzt/juv) uses for notebooks). The
viewer reads the block and prepends a `%pip install` cell, so activating and
running installs everything before the rest of the document executes.

```{code-cell} python
# /// script
# requires-python = ">=3.12"
# dependencies = [
#     "pyjokes==0.8.3",
#     "numpy==2.5.1",
# ]
# ///
```

`pyjokes` is not bundled with Pyodide — it only works here because the metadata
above got it installed:

```{code-cell} python
import pyjokes

print(pyjokes.get_joke())
```

Declared dependencies that Pyodide already ships (like `numpy`) are satisfied
instantly:

```{code-cell} python
import numpy as np

print(np.__version__)
```
